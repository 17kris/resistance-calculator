Welcome to my resistance calculator.

All towers need to be able to determine the resistance that a vehicle will create during the recovery process. This is a critical part of the safe and successful recovery.

powerful calculator

Using a sophisticated algorithm this program will simplify the process as well as increasing the accuracy.

Increase to operators safety

Through the use of this calculator, operators can work within their equipment's safe working load limits, while not having to struggle with over-sized rigging.
Gone, will be the days when operators say: Hope this works! This should do it! Let's try this!
